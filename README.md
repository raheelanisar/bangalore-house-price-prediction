# 🏠  House Price Prediction
# Task 6
## Objective
Predict house prices in Bangalore based on features like location, 
square footage, BHK, and bathrooms.

## Dataset
Bangalore House Price Dataset (Kaggle)

## Models Applied
- Linear Regression
- Data preprocessing with outlier removal & feature engineering

## Tech Stack
- Python, Pandas, NumPy, Scikit-learn
- Flask (Web App)
- HTML/CSS (Frontend)

## Key Results
- Achieved good accuracy using Linear Regression
- Deployed as a working Flask web application

## How to Run
1. Clone the repo
2. Install requirements: `pip install -r requirements.txt`
3. Run: `python app.py`
